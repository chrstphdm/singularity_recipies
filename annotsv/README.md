# AnnotSV : Annotation and Ranking of Human Structural Variations

Annot is a program designed for annotating and ranking Structural Variations (SV). 
This tool compiles functionally, regulatory and clinically relevant information and aims at providing annotations useful to 
i) interpret SV potential pathogenicity
ii) filter out SV potential false positives.

## Source code

<https://github.com/lgmgeo/AnnotSV>

## Usage

```bash
$ANNOTSV/bin/AnnotSV -SVinputFile yourFile.bed -outputFile ./test.annotated.tsv -svtBEDcol 4 >& AnnotSV.log &
```
